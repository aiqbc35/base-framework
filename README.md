# 基础框架

> 利用各种类库组合的一个框架


#### 类库
-------------

    - illuminate/routing 路由类
    - illuminate/events  事件类
    - illuminate/database  ORM数据库操作
    - illuminate/view   视图类
    - filp/whoops   错误提示类
    - symfony/var-dumper,   打印信息 美化版的var_dump
    - monolog/monolog,   日志类
    - vlucas/phpdotenv   读取.env文件类
